# keralabuses.in

A modern, community-driven web application to track Private and KSRTC bus timings across Kerala.

## Tech Stack
* **Frontend:** React.js (Vite), Lucide Icons
* **Styling:** Tailwind CSS (Responsive Design)
* **Backend/DB:** Firebase Firestore (Real-time updates)
* **Hosting:** Firebase Hosting

## Key Features
* **Smart Search:** Filter buses by Origin, Destination, or Stops.
* **Community Powered:** Users can upvote reliable buses, add comments, and report delays.
* **Live Status:** "Late" and "On Time" indicators based on user feedback.
* **Fare Calculator:** Built-in tool to estimate ticket prices based on kilometers.
* **Dynamic Routing:** Hash-based routing for easy link sharing.
